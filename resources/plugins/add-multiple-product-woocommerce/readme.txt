== Description ==

The 'Add Multiple Products To Cart' WooCommece plugin makes shopping a one-click process and works perfectly in cases where customization of products or their quantities is involved.

== Installation ==

1. Upload the entire 'add-multiple-product-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Please enable the Permalinks from admin under the Setting->Permalinks
